# Loom Builder Dockerfile

A Dockerfile for Ubuntu 16.04 with the following packages:

 * apt-utils
 * gcc
 * cmake
 * libuv1-dev
 * libprotobuf-dev
 * python3-dev
 * python3-pip

and the following Python 3 packages:

 * cloudpickle
 * protobuf
 * pytest
 * pytest-timeout
 * bokeh
 * matplotlib
 * sphinx-rtd-theme
 * flake8
